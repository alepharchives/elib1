Validation

XML and HTML are validated with the command:

      > xml_check <File>

xml_check is as follows:

#!/bin/sh
export XML_CATALOG_FILES=$HOME/code/elib2-1/dtd/catalog.xml
unset SGML_CATALOG_FILES
xmllint  --nonet --valid --noout  $1

The environemnt variable must point to this directory.

catalog.xml also contains some absolut paths which must be edited to reflect the absolute location of the DTD files.
